
speech_to_txt.py : converte in testo un audio nei vari formati .m4a/.mp3/.3ga/.wav

n.b.: installare :
           ffmpeg da https://www.gyan.dev/ffmpeg/builds/
           > pip install audioop-lts
           > pip install SpeechRecognition

lanciare l'applicazione con :
           > py speech_to_txt.py

si aprirà una finestra di dialogo dove sarà possibile
scegliere il tipo di file audio da caricare e convertire
in testo.
